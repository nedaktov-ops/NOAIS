# NOAIS v1.0.0 — 2026-06-07

**First public release.** 199/199 tests passing, CI green on GitHub Actions, Apache-2.0.

NOAIS ("No Artificial Intelligence Slop") is a free, open-source browser extension that helps you detect, mask, and disable AI-generated text on the web. It's 100% on-device — no network calls, no accounts, no telemetry, no AI model. It uses stylometric heuristics (burstiness, type-token ratio, Shannon entropy, hapax ratio) plus a hard-coded AI-phrase counter to flag text that looks machine-generated.

## What's included

- **4 platform adapters:** YouTube, Facebook, Instagram, TikTok.
- **Stylometric AI-likely scoring** (0–100) for every comment / post / caption.
- **Per-site + global sensitivity** (0–100 slider).
- **Soft mode** (default): small badge + colour-coded severity outline.
- **Hard mode** (per-site toggle): dim + blur with hover-reveal.
- **MutationObserver** for infinite scroll on all 4 platforms.
- **Light + dark mode** UI (follows browser preference).
- **Options page** with the full settings UI.
- **Zero network calls. Zero telemetry. Zero accounts.**

## Install

### Chrome / Chromium / Brave / Edge

1. Download `NOAIS-v1.0.0-chrome.zip` from this release (below).
2. Unzip it.
3. Open `chrome://extensions` in your browser.
4. Enable **Developer mode** (top right).
5. Click **Load unpacked** and select the unzipped directory.

### Firefox (manual install)

1. Download `NOAIS-v1.0.0-firefox.zip` from this release (below).
2. Open `about:addons` in Firefox.
3. Click the **gear icon** (top right) → **Install Add-on From File...** → select the zip.

The Firefox AMO listing is in review. See [`EXTENSION-LISTING.md`](https://github.com/nedaktov-ops/NOAIS/blob/main/EXTENSION-LISTING.md) for the pre-formatted submission metadata.

## What's NOT in v1.0

- **Chrome Web Store listing.** Requires a $5 one-time developer fee. The user can submit the same `NOAIS-v1.0.0-chrome.zip` to the CWS at any time.
- **iOS Safari.** Out of scope (we develop on Linux, no iOS toolchain).
- **Auto-update.** NOAIS has no update server; re-download from GitHub Releases for new versions.
- **ONNX model.** v0.8 was genuinely optional; the heuristics already catch the easy cases. A future v1.1+ may revisit.

## Known issues

- The background service worker logs "installed (reason: install)" on every Chromium cold start. This is cosmetic — the SW is registered once but logs every wakeup. Tracked for v1.0.1.
- Hard mode is global per site (no per-element override). If you want to keep some elements un-blurred, you'll need to add a per-element allowlist; that's a v1.1 feature.
- The popup "On this site" toggle is sync only — there's no per-tab persistence. If you toggle off on a tab and reload, the toggle persists. v1.1 will add per-tab state.
