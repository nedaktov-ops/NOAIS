# NOAIS v1.1.0 — 2026-06-08

**Settings + UI + i18n + sync.** 277/277 tests passing, CI green on GitHub Actions, Apache-2.0.

## What's new

- **First-run welcome page** — explains NOAIS, permissions, and how to use it. Opens automatically on first install.
- **"Why am I seeing this?" side panel** — click the "Why?" link in any badge to open Chrome's side panel and see the full breakdown (burstiness, type-token ratio, Shannon entropy, hapax ratio) for that specific element.
- **Keyboard shortcut** — Ctrl+Shift+A toggles NOAIS on/off for the current site.
- **`chrome.storage.sync`** — `noais_enabled` and `noais_global_sensitivity` sync across your Chrome profile. `noais_hard_mode_sites` stays local (content script only accesses local).
- **Hard mode sites card** — per-site hard mode toggle in the options page. Hard mode = dim + blur.
- **Per-tab disable button** — "Disable on this site" in the popup. Persists per-tab override.
- **60+ i18n strings** — full English localisation for all UI text.
- **Page counter** — shows AI phrase count + score on every page, draggable, expandable.
- **Badge tooltip** — hover any NOAIS badge to see score breakdown + "Allowlist this text" + "Don't show this element".
- **Element allowlist** — right-click any badge → "Allowlist this text" removes the badge permanently.
- **Headless fixtures** — welcome page, why panel, page counter, element allowlist all have headless Chromium test fixtures.
